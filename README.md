# Sashakt Rashtra Nirman (SRN) Project Workspace

Welcome to the central repository for the **Sashakt Rashtra Nirman (SRN)** platform. This workspace contains the complete backend infrastructure, technical documentation, and project requirements for the SRN NGO platform.

---

## Repository Structure

This repository is organized into the following key areas:

### 1. [Backend Services](./artifacts/srn-backend)
The core API infrastructure built with Node.js, TypeScript, and PostgreSQL. It includes:
- **Authentication**: JWT & Google OAuth 2.0.
- **Documentation**: Swagger UI integration.
- **DevOps**: Docker and CI/CD pipelines.
- **Testing**: Automated Jest/Supertest suite.

### 2. [Project Documentation](./SRN_Project_Documents)
Official project specifications and requirements:
- **PRD**: Product Requirement Document.
- **TRD**: Technical Requirement Document.
- **UI/UX**: Design briefing and application flow.
- **Schema**: Backend database implementation details.
